# Contributing to TRIO

Thanks for your interest in contributing! 🚀

## 🧭 How to Contribute
1. Fork the repository
2. Create a new branch (`feature/your-feature`)
3. Commit your changes
4. Open a Pull Request

## ✅ Commit Style
Use short, clear messages:
```
feat: add focus streaks
fix: timer pause bug
ui: improve dashboard layout
```

## 🧪 Testing
Run locally:
```bash
flutter test
```

## 💡 Ideas
Feel free to open an Issue with feature ideas or improvements.

---

Happy building! 💜
